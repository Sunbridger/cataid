## 🎯 两个入口的功能

### 1. 我领养的猫猫
- **路由**: `/my/cats`
- **图标**: 🐱 Cat (琥珀色)
- **描述**: 查看已成功领养的毛孩子
- **功能**: 显示用户成功领养的所有猫咪

- **路由**: `/my/published`
- **图标**: ❤️ Heart (蓝色)

## 🔧 技术实现

### MenuItem 组件增强

```typescript
const MenuItem: React.FC<{
  icon: React.ReactNode;
  title: string;
  description?: string;  // 新增
  color?: string;
  to?: string;
  onClick?: () => void;
  badge?: string;
}> = ({ icon, title, description, color, to, onClick, badge }) => {
  // ...
  <div className="flex-1">
    <div className="font-bold text-slate-700 text-sm">{title}</div>
    {description && (
      <div className="text-slate-400 text-xs mt-0.5">{description}</div>
    )}
  </div>
  // ...
};
```

### 使用方式

```tsx
<MenuItemGroup title="我的猫咪">
  <MenuItem
    icon={<Cat size={18} />}
    title="我领养的猫猫"
    description="查看已成功领养的毛孩子"
    color="text-amber-500"
    to="/my/cats"
  />
  <MenuItem
    icon={<Heart size={18} />}
    title="我发布的猫猫"
    description="管理我发布的领养信息"
    color="text-blue-500"
    to="/my/published"
  />
</MenuItemGroup>
```
